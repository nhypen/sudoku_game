# 🧩 Sudoku Game

Prosta gra Sudoku stworzona jako projekt do nauki i wrzucona na GitHub Pages.

## 🚀 Funkcje

* Generowanie nowych plansz
* 3 poziomy trudności: łatwy, średni, trudny
* Sprawdzanie poprawności
* Podświetlanie wierszy, kolumn i sekcji 3x3
* Możliwość pokazania rozwiązania

## 🛠️ Technologie

* HTML
* CSS
* JavaScript

## ▶️ Jak uruchomić

1. Pobierz repozytorium
2. Otwórz plik `index.html` w przeglądarce
